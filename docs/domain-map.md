# Domain Map

## Core Surfaces
- Public website: marketing, teacher branding, package highlights, future acquisition funnel.
- Student portal: future student-facing shell, purchases, content access, support entry points.
- Admin dashboard: Milestone 1 operational control surface for auth, permissions, settings, academic taxonomy, and auditability.

## Active Milestone 1 Domains
- `Identity`
  - Admin authentication
  - Admin and manager records
  - Roles and permissions
  - Settings registry
  - Audit logs
  - Admin dashboard
- `Academic`
  - Grades
  - Tracks
- `Shared`
  - Contracts
  - Navigation
  - Exports
  - UI components
  - Shared enums

## Reserved Future Domains
- `Students`
  - Student auth scaffold
  - Student entity base for CRM and portal work
- `Commerce`
  - Products, orders, entitlements, shipping
- `Centers`
  - Centers, groups, attendance
- `Support`
  - Complaints, suggestions, tickets
- `Operations`
  - Shifts, payroll, internal operations
